## 用語紹介
セレクションバイアス
> 比較しているグループの潜在的な傾向(その他の要素)が違うことによって発生するバイアス。eg:メール配信の分析

RCT(無作為比較試験)
> 施策の割り当てをランダムにし、介入が行われたグループと介入が行われていないグループの比較を可能にする。"因果推論の根本問題"を限りなく解消する。eg:ABテスト
